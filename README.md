# TaskManager
This is a Task Manager written in Python using PyQt5.


## Features
```
1) User can manage all the system processes like viewing, suspending and killing the processes.
2) User can monitor the CPU usage.
3) User can monitor the memory used by the system.
4) User can monitor the network speed and other details of the system.
```
## Installation and Working
```
1) Install python3 and above using the link:
        - https://realpython.com/installing-python/
        
2) Clone the project in a directory using the link:
        - https://confluence.atlassian.com/bitbucket/clone-a-repository-223217891.html
   Or Download the Zip file of the project.
        
3) Create a virtual environment using the link
        - https://packaging.python.org/guides/installing-using-pip-and-virtual-environments/
        
3) Install the requirements from requirements.txt using:
        - pip install -r requirements.txt
        
4) Run file main.py:
        - python3 main.py
```

## Build
```
* Build can be made using the commands written below:
        - pip install pyinstaller
        - pyinstaller --onefile main.py
        
  Open the "dist" folder and click on the "main" file.
  
* The build will be OS specific i.e build for Unix will work on Unix systems only.
```
